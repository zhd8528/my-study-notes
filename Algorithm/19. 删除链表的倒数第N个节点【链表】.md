# 19. 删除链表的倒数第N个节点【链表】

> 给你一个链表，删除链表的倒数第 n 个节点，并且返回链表的头节点

![img](https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg)

**题解**：

```java
  class Solution{
      public ListNode removeNthFromEnd(ListNode head, int n){
          //虚拟头节点
          ListNode dummy = new ListNode(0);
          dummy.next = head;
          ListNode slow = dummy;
          ListNode fast = dummy;
          //快指针先走 n + 1 步
          for(int i = 0; i <= n; i++){
              fast = fast.next;
          }
          //快慢同走，直到 fast 为空
          while(fast != null){
              slow = slow.next;
              fast = fast.next;
          }
          //删除倒数第 n 个
          slow.next = slow.next.next;
          return dummy.next;
      }
  }
```

> 核心思路：
>
> 快慢指针 + 虚拟头节点
>
> 1. 新建虚拟头节点 dummy，指向原头，规避删除头节点的边界问题
> 2. 快指针先走 n +  1 步
> 3. 快慢指针一起往后走，直到快指针走到末尾
> 4. 此时慢指针正好在待删除节点的前驱
> 5. 慢指针直接跳过下一个节点：slow.next = slow.next.next
>
> 口诀：
>
> 删倒数 n 用双针，虚拟头节点先蹲；
>
> 快针先走 n 加一，同走到尾删后继
