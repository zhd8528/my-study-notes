# 70. 爬楼梯【动态规划】

> 假设你正在爬楼梯
>
> 需要 n 阶你才能到达楼顶
>
> 每次你可以爬 1 或 2 个台阶
>
> 你有多少种不同的方法可以爬到楼顶呢？

**错误示例**（思路错误）：

```java
class Solution{
    public int climbStairs(int n){
        if(n <= 3) return n;
        int a = 1, b = 1;
        int sum = 0;
        for(int i = 0; i < n; i++){
            sum = a + b;
            a = b;
            b = sum;
        }
        return b;
    }
}
```

**正解**：

```java
class Solution{
    public int climbStairs(int n){
        if(n <= 2) return n;
        int a = 1, b = 2;
        int sum = 0;
        for(int i = 3; i <= n; i++){
            sum = a + b;
            a = b;
            b = sum;
        }
        return sum;
    }
}
```

> 核心思路：
>
> 动态规划，本质斐波那契数列
>
> 第 n 阶只能从 n - 1阶走 1 步或 n - 2 阶走 2 步
>
> 状态转移：
>
> dp[n] = dp[n - 1] + dp[n - 2]
>
> 初始条件：
>
> dp[1] = 1
>
> dp[2] = 2
>
> 优化：
>
> 不用数组，只用两个变量滚动迭代，省空间
>
> 口诀：
>
> 一阶一种两种二，后面全靠前俩和；
>
> 滚动变量省空间，递推一遍出结果
