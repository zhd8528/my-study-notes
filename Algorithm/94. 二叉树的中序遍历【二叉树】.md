# 94. 二叉树的中序遍历【二叉树】

> 给定一个二叉树的根节点`root`，返回它的中序遍历

![img](https://assets.leetcode.com/uploads/2020/09/15/inorder_1.jpg)

**先定义二叉树**：

```java
public class TreeNode{
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(){}
    TreeNode(int val){
        this.val = val;
    }
    TreeNode(int val, TreeNode left, TreeNode right){
        this.val = val;
        this.left = left;
        this.right = right;
    }
}
```

**题解**（递归）：

```java
class Solution{
    List<Integer> res = new ArrayList<>();
    public List<Integer> inorderTraversal(TreeNode root){
        inOrder(root);
        return res;
    }
    
    private void inOrder(TreeNode node){
        if(node == null) return;
        //根
        inOrder(node.left);
        //左
        res.add(node.val);
        //右
        inOrder(node.right);
    }
}
```

**错误示例**（栈越界）：

```java
class Solution{
    public List<Integer> inorderTraversal(TreeNode root){
        List<Integer> res = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        TreeNode node = root;
        while(node != null){
            while(node.left != null){
                stack.push(node);
                node = node.left;
            }
            node = stack.pop();
            res.add(node.val);
            node = node.right;
        }
        return res;
    }
}
```

**正解**（迭代）：

```java
class Solution{
    public List<Integer> inorderTraversal(TreeNode root){
        List<Integer> res = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        TreeNode node = root;
        while(node != null  || !stack.isEmpty()){
            //一直往左走，全部入栈
            while(node != null){
                stack.push(node);
                node = node.left;
            }
            //此栈访问根
            node = stack.pop();
            res.add(node.val);
            //转向右子树
            node = node.right;
        }
        return res;
    }
}
```

> 核心思路：
>
> 中序遍历固定顺序：
>
> 左 -> 根 -> 右
>
> 两种写法：
>
> 递归版（最简单）、迭代栈版（面试常问）
>
> 口诀：
>
> 中序遍历左根右，递归无脑往下走；
>
> 迭代栈来存路径，先压左树再出头
