# 53. 最大子数组和【动态规划】

> 给你一个整数数组 `nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和
>
> 子数组是数组中的一个连续部分

**错误案例**（没有考虑元素可能为负数）：

```java
class Solution{
    public int maxSubArray(int[] nums){
        int pre = 0, max = 0;
        for(int num : nums){
            pre = Math.max(pre + num, num);
            max = Math.max(max, pre);
        }
        return max;
    }
}
```

**正解**：

```java
class Solution{
    public int maxSubArray(int[] nums){
        int pre = nums[0], max = nums[0];
        for(int i = 1; i < nums.length; i++){
            //重新开始或接续前值
            pre = Math.max(pre + nums[i], nums[i]);
            //更新全局最大值
            max = Math.max(max, pre);
        }
        return max;
    }
}
```

> 核心思路：
>
> 贪心/动态规划（kadane 算法）
>
> 1. 维护两个变量：
>    - pre：以当前元素结尾的最大子数组和
>    - max：全局记录最大子数组和
> 2. 遍历每个元素：
>    - pre = max(当前元素, pre + 当前元素)
>    - 要么自己单独开始，要么接前面的子数组
>    - 每次更新 max 保留最大值
>
> 口诀：
>
> 能加就接着加，不如从头就放下，全局最值随时抓
